📚 Digital Library Management System
📌 Project Overview

The Digital Library Management System is a simple and user-friendly application built to manage library books digitally.
It allows users to add, view, update, delete, and search books efficiently using a secure login system.

This project is designed to demonstrate basic CRUD operations, database handling, and a simple user interface.

🛠️ Technologies Used

Python

SQL (Database)

Streamlit (Frontend/UI)

✨ Features

🔐 User Authentication

➕ Add Book

📖 Display Books

✏️ Update Book

🗑️ Delete Book

🔍 Search Book

❌ Exit Application

📘 Book Details (Sub-sections)

Each book contains the following information:

Book ID

Book Name

Author Name

These fields are used across all operations like add, update, search, and delete.

🔐 Security Feature

🚀 How to Run the Project

Install required libraries:

Run the application:

streamlit run app.py
